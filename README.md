# The Mountain

A 3D survival game developed in Unity and C# that challenges players to gather resources, manage survival needs, and navigate a dynamic environment.

## Features

* Inventory management system
* Resource gathering mechanics
* Day and night cycle
* Real-time player stat tracking
* Physics-based interactions
* Modular HUD and UI systems
* Responsive player controls using Unity's New Input System

## Technologies Used

* Unity
* C#
* Unity Input System

## Gameplay Systems

### Inventory System

Allows players to collect, store, and manage resources throughout gameplay.

### Resource Gathering

Players can gather materials from the environment to support survival and progression.

### Day/Night Cycle

A dynamic environmental system that changes the game world over time.

### Player Statistics

Tracks important survival metrics and updates the user interface in real time.
